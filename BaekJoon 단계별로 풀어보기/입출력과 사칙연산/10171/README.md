# 고양이 문제

## 문제

> 아래 예제와 같이 고양이를 출력하시오.

## 출력

```
\    /\
 )  ( ')
(  /  )
 \(__)|
```

### 문제 풀이

테스트입니다.

```java
public class Cat {
    public static void main(String[] args) {

        System.out.println("\\    /\\");
        System.out.println(" )  ( ')");
        System.out.println("(  /  )");
        System.out.println(" \\(__)|");

    }
}
```
