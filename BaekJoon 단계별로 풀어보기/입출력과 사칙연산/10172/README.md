# 개 문제

## 문제

> 아래 예제와 같이 개를 출력하시오.

## 출력

```
|\_/|
|q p|   /}
( 0 )"""\
|"^"`    |
||_/=\\__|
```

### 문제 풀이

/와 "를 입력할시엔 /를 앞에 붙여주세요.

```javascript
public class dog {

    public static void main(String[] args) {

        System.out.println("|\\_/|");
        System.out.println("|q p|   /}");
        System.out.println("( 0 )\"\"\"\\");
        System.out.println("|\"^\"`    |");
        System.out.println("||_/=\\__|");
    }
}
```
